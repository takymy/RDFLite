# RDFLite [COMP2322 Coursework]
## About
RDFLite is a domain specific programming language for querying simple RDF 
documents. RDF - the Resource Description Framework - is a data model and 
syntax for describing graphs, and is the underlying technology used on the 
Semantic Web. Was developed as a part of Programming Language Concepts 
coursework.
